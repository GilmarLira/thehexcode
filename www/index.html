Unravel the mysteries of the HEX code

This website is an attempt to break down the mysteries of color reproduction on the web through the understanding of display technology and demystifying the hexadecimal color code.

Get started!

Display Technology

Those pixels on your screen carry a vast amount of color information. Each pixel of color works because of a controlled variation of voltage that lets each pixels range from 256 shades of red, green or blue light. That’s 16.8 million colors! woow!

Additive Color Model

This is how colors are generated on a screen! It works by using variable light intensity with red, green and blue colors. Abscence of light equals black. The more light you add the lighter/whiter the results are.
Go on, give it a try!

Color In Code

Computers communicate in a binary language, each color has a series of digits associated to it. This series of numbers and letters is called HEX - or Hexadecimal, usually represented by the # sign. It’s an awesome method you can use to quickly change the colors on your website.
