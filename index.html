An Introduction

Unravel the mysteries of the HEX code

The colors we use in web design have the power to turn emotions in the blink of an eye. With such great power, we must exhibit great care in their use. As technology progresses, so do the tools for harnessing this power. These tools might seem complicated at first, but it’s not so bad and not really that complicated at all.

This website is an attempt to break down the mysteries of color reproduction on the web through the understanding of display technology and demystifying the hex code.
